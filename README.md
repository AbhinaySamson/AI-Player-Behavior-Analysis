# AI-Based Player Behavior Analysis System

## 📌 Project Overview
This project is a web-based application that analyzes player behavior using gameplay data and machine learning. It evaluates how users perform in different games and provides insights into their decision-making and cognitive skills.

---

## 🎮 Games Included
- Tic Tac Toe → Strategic thinking  
- Grid Path → Path optimization  
- Memory Game → Memory & recall  
- NeuroSprint → Reaction speed  

---

## 🧠 Key Features
- Per-game behavior analysis (main innovation)
- Machine learning-based classification
- Performance insights (accuracy, efficiency, mistakes)
- Interactive dashboard with graphs
- Best game and weak game identification

---

## 🤖 Machine Learning Used
- K-Means Clustering → Behavior classification  
- PCA → Data visualization  
- Linear Regression → Performance prediction  

---

## ⚙️ Tech Stack
- Frontend: React.js + Tailwind CSS  
- Backend: Flask (Python)  
- Database: MongoDB  
- ML Libraries: Scikit-learn, NumPy, Pandas
